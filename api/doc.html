Light documentation

Written 31 Dec 2014 by Jarkko Moilanen

The API provides JSON listings of data related to lentodiilit.fi service. The database is updated hourly. In other words, there are moments when you will not get the most recent listing about the situation.

About implementation

Database update is implemented with Ruby program which scrapes the data from lentodiilit.fi pages. API is build on top of Epiphany PHP Framework. It is a micro PHP framework that's fast, easy, clean and RESTful. The framework does not do a lot of magic under the hood. It is, by design, very simple and very powerful. https://github.com/jmathai/epiphany

Unofficial experimental Open API

Furthermore, this API is unofficial. API is not part of the lentodiilit.fi service and is maintained by Jarkko Moilanen. This API is open API. Source code for the API is in Github.

/deals

  • Returns JSON listing of 100 (max maybe less) last deals
  • Order descending == latest first
